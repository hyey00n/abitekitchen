🎯 에이바이트키친 마케팅 추적 코드

아래 코드들을 HTML 파일의 <head> 섹션에 복사해서 붙여넣으세요.

1. Google Analytics 4 (GA4) - 필수 ✅

설치 위치: 모든 페이지의 <head> 태그 안, 가장 위쪽

⚠️ 설치 전 준비:
1. Google Analytics 접속
2. 계정 생성 → 속성 생성 → 데이터 스트림 추가
3. 측정 ID (G-XXXXXXXXXX) 복사
4. 아래 코드의 G-XXXXXXXXXX를 본인의 측정 ID로 변경 
<!-- Google Analytics 4 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
  
  // 커스텀 이벤트 - QR 코드 스캔 추적
  if (window.location.search.includes('utm_source=qr')) {
    gtag('event', 'qr_scan', {
      'event_category': 'engagement',
      'event_label': 'business_card'
    });
  }
  
  // 커스텀 이벤트 - 제휴문의 버튼 클릭 시
  function trackPartnershipClick() {
    gtag('event', 'partnership_inquiry', {
      'event_category': 'conversion',
      'event_label': 'form_click'
    });
  }
</script>

제휴문의 버튼에 추가:

<a href="구글폼링크" onclick="trackPartnershipClick()">제휴 문의하기</a>

2. Meta Pixel (페이스북/인스타그램) - 선택 ⭕

설치 위치: 모든 페이지의 <head> 태그 안

⚠️ 설치 전 준비:
1. Meta Events Manager 접속
2. 픽셀 생성
3. 픽셀 ID (숫자) 복사
4. 아래 코드의 YOUR_PIXEL_ID를 본인의 픽셀 ID로 변경

3. UTM 파라미터 전략 🎯

유입 경로별 URL을 구분해서 사용하세요. GA4에서 소스별 분석이 가능합니다.

용도 URL 설명
명함 QR코드 https://abitekitchen.com/?utm_source=qr&utm_medium=business_card&utm_campaign=2026_branding 명함 효과 측정
인스타 프로필 https://abitekitchen.com/?utm_source=instagram&utm_medium=bio_link&utm_campaign=social_organic 인스타그램 유입
인스타 스토리 https://abitekitchen.com/?utm_source=instagram&utm_medium=story&utm_campaign=social_organic 스토리 링크 클릭
이메일 서명 https://abitekitchen.com/?utm_source=email&utm_medium=signature&utm_campaign=email_signature 이메일 서명란
기본 URL https://abitekitchen.com/ 파라미터 없음 (직접 입력)
💡 QR 코드 생성 방법:
1. QR Code Generator 접속
2. 위 UTM URL 복사해서 입력
3. QR 코드 다운로드
4. 명함 디자인에 삽입

4. Google Search Console - 선택 ⭕

설치 위치: <head> 태그 안

⚠️ 설치 전 준비:
1. Google Search Console 접속
2. 속성 추가 → URL 접두어 선택
3. HTML 태그 확인 방법 선택
4. 메타 태그 복사
5. 아래 코드 형식으로 붙여넣기 
<!-- Google Search Console 소유권 확인 -->
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />

✅ 설치 체크리스트

  • ☐ Google Analytics 계정 생성 및 측정 ID 발급
  • ☐ GA4 코드 모든 페이지에 설치
  • ☐ Meta Pixel 생성 (인스타 활용 시)
  • ☐ UTM URL 생성 (명함 QR, 인스타 프로필)
  • ☐ QR 코드 생성 (UTM URL 포함)
  • ☐ Google Search Console 소유권 확인
  • ☐ 실제 방문 테스트 (모바일/데스크톱)
  • ☐ GA4 실시간 보고서에서 데이터 확인

📊 GA4에서 확인하는 방법

  1. Google Analytics 접속
  2. 보고서 → 실시간 보고서 (방금 들어온 방문자 확인)
  3. 보고서 → 획득 → 트래픽 획득 (유입 소스별 분석)
    • qr / business_card (명함 QR)
    • instagram / bio_link (인스타 프로필)
    • (direct) / (none) (직접 입력)
  4. 보고서 → 참여 → 페이지 및 화면 (페이지별 체류시간)
  5. 보고서 → 참여 → 이벤트 (커스텀 이벤트 확인)
    • qr_scan (QR 스캔)
    • partnership_inquiry (제휴문의 클릭)

💡 주요 팁

  • 데이터 쌓는 기간: 최소 1개월 데이터 필요 (신뢰도 높은 분석)
  • 월간 리포트: 매월 1일에 GA4 확인 후 간단히 정리
  • 투자자용 자료: "월 500명 방문, QR 60% 유입, 제휴문의 5건" 같은 데이터
  • 모바일 테스트: 실제 스마트폰으로 QR 스캔 후 GA4 실시간 확인
